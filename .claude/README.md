# Claude Code 作業履歴

このフォルダには Claude Code での作業計画・履歴を保存しています。

## フォルダ構成

```
.claude/
├── README.md           # このファイル
└── sessions/           # 作業セッションの記録
    └── YYYY-MM-DD-{task-name}.md
```

## 命名規則

- `YYYY-MM-DD-{task-name}.md`
- 例: `2026-01-21-improve-specs.md`

## 用途

- AI エージェント（Claude Code, Google Antigravity 等）での作業計画
- 作業内容の履歴・参照
- 将来の類似タスクでのコンテキスト提供
