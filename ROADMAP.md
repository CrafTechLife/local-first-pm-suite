## Phase 1.5: マスタ管理・機能拡張 (Current Focus)
**v2.0 目標**

### 優先度：高
これまでのJSON直接編集や文字列入力を廃止し、管理機能としての体裁を整える。

- [ ] **カテゴリ管理** (0.5日)
  - [ ] カテゴリ定義（ID, 名前, 色）の追加・編集
  - [ ] WBS項目へのカテゴリ紐付け
- [ ] **メンバー管理** (0.5日)
  - [ ] メンバー定義（ID, 名前, 役割）の追加・編集
  - [ ] 担当者アサインのID管理化（複数人対応）
- [ ] **マイルストーン専用管理** (1日)
  - [ ] 専用JSONファイルへの分離
  - [ ] マイルストーン管理UI
  - [ ] WBSツリーからの表示分離

### 優先度：中
プロジェクト管理の精度と視認性を向上させる。

- [ ] **ステータス拡張 (6段階)** (0.5日)
  - [ ] `not-started` (未着手)
  - [ ] `on-track` (順調)
  - [ ] `at-risk` (注意)
  - [ ] `delayed` (遅延)
  - [ ] `on-hold` (保留)
  - [ ] `completed` (完了)
- [ ] **ガントチャート機能拡張** (1日)
  - [ ] ズーム機能（日/週/月スケール）
  - [ ] 現在地（今日）へのジャンプ
- [ ] **フィルタリング機能** (0.5日)
  - [ ] 担当者別
  - [ ] カテゴリ別
  - [ ] ステータス別

### 優先度：低
- [ ] **設定メニューUI** (0.5日)
  - 各種マスタ管理へのアクセスポイント作成
