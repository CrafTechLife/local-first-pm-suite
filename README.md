# WBS Manager 開発・利用ガイド

このプロジェクトは、Windowsの業務PCで「インストール不要」で動作する WBS 管理ツールです。

## 📁 ディレクトリ構成

- `wbs-manager.html`: **[成果物]** 運用・配布用ファイル。ブラウザでそのまま開けます。
- `src/`: 開発用の分割ソースコード。
  - `index.html`: 構造のテンプレート。
  - `main.js`: メインロジック。
  - `js/`: 各種機能モジュール（保存、状態管理、UI描画など）。
  - `styles/`: スタイルシート。
- `build.js`: ソースコードを1つのHTMLに統合するビルドスクリプト。

## 🚀 使い方

### 1. 利用者（配布・運用）
- `wbs-manager.html` を Chrome や Edge などのブラウザで開いてください。
- 配布する際は、このファイル1つを渡すだけでOKです。

### 2. 開発者（コードの修正）
メンテナンス性を高めるため、ソースコードは `src/` 配下に分割されています。

1. `src/` 配下の JS や CSS を編集します。
2. 以下のコマンドを実行して、変更を `wbs-manager.html` に反映（ビルド）させます。
   ```bash
   node build.js
   ```

## 🛠️ Windows環境での開発
Node.js がインストールできない環境で開発を行う場合は、以下のコマンド（macOS標準）の代わりに、ポータブル版の Node.js や代替ツールを使用してビルドを行ってください。
※通常は配布された `wbs-manager.html` を利用するだけなので、ビルドは開発時のみ必要です。
